
Creating a new Alya related project in the repository machine:
--------------------------------------------------------------

- type in the repository machine:
cd /gpfs/projects/bsc21/svnroot
mkdir AlyaNewProject

- create there svn structure:
svnadmin create /gpfs/projects/bsc21/svnroot/AlyaNewProject

- now from your local machine, step on the AlyaNewProject upper folder 
and type:

svn import AlyaNewProject svn+ssh://bsc21***@mn2.bsc.es/gpfs/projects/bsc21/svnroot/AlyaNewProject/Trunk

and listo calisto.
